Website building resources


HTML

- W3schools - Free coding tutorials.
- Codepen - Live code editor that shows you changes in real-time. Handy for testing code snippets.
- A Field Guide to Web Accessibility - In an age where more people use the internet than ever, making your site accessible is crucial. This is a good article to read if you don't know where to start.
- HTML Cheatsheet - Resource compiling HTML/CSS assets, layout generators, code snippets, and more.

Other sites I love

- The Internet Archive - A massive database archiving over 880 billion webpages.
- Archiverse - Archive of over 539 million posts, replies, profiles, avatars, communities and more from Nintendo's now-defunct social media service, Miiverse. (That's 17 terabytes of data!)
- Logan Whitehurst's Official Website of Luv and Stuff - The personal website of the late Logan Whitehurst, a talented visual artist and musician.
- Toyhouse - A site for storing original characters, complete with a forum and allowance for roleplaying, worldbuilding and trading. (You need an invite code to join, but those are really easy to find. Email me if you want one, I have a lot left over!)
- Artfight - An annual art-trading game taking place every July.
- The Lemonade Stand - A neat archive of Lemon Demon/Neil Cicierega related stuff.
